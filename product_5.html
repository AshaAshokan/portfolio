About app

In this case study, we will explore the process I have followed to improve the mobile site (msite) by transitioning to a bloks framework, aiming to provide a more intuitive and engaging mobile user experience. I'll examine the steps taken, the challenges faced, and the results achieved.

While we've worked on multiple surfaces as part of this comprehensive project, this case study will provide an in-depth analysis of the transformation of the edit profile screen to deliver a better mobile user experience.

Problem Statement

Challenge: The existing edit profile screen on our msite faced usability issues, impacting users' ability to navigate and update their information. While this case study zooms in on the edit profile screen, it's important to note that it's part of a broader initiative to improve multiple screens within the framework.

Objective: This redesign aimed to address usability issues, enhance performance, and provide a seamless editing process. While we have worked on many screens, the focus here is on the edit profile screen as an illustrative example.

Goals

  • Improved Usability: To create a user-friendly edit profile screen that's intuitive and easy to use.
  • Performance Enhancement: Optimize the performance of the screen to ensure it loads quickly and provides a responsive experience.
  • Visual Consistency: Ensure that the screen design maintains visual coherence within the framework's block system.
  • Streamlined Editing Process: Simplify the process of updating user information.

Research

------ Usability Test

Conducted usability testing sessions with 15 participants. Tasks included updating profile information, changing profile pictures, and adjusting privacy settings. Observed user behaviour, noted task completion times, and recorded any difficulties.

------ Heatmaps and Click Tracking:

Utilized analytics tools to generate heatmaps and click-tracking data. Identified areas with low engagement and user interaction on the edit profile screen.

User Feedback and Pain Points (Qualitative Data):

Users reported confusion regarding the arrangement of profile information, with comments such as "Information was scattered, hard to find." Many users expressed frustration with the previous screen, noting, "Updating my profile felt cumbersome and time-consuming." Users frequently mentioned that the old design lacked visual consistency and alignment with the rest of the mobile site.

Usability Test Observations:

During usability testing, users were observed hesitating and making multiple attempts to edit their profile information. Users often commented on the cluttered appearance of the old screen, stating that it was "overwhelming." Several users mentioned that the lack of clear labels and intuitive icons made it difficult to understand the purpose of certain elements.

Performance Metrics (Quantitative Data):

The average page load time for the old edit profile screen was recorded at 4.5 seconds. Task completion rates for updating profile information were measured at 65% before the redesign. User satisfaction scores for the old screen averaged at 2.7 on a scale of 1 to 5..

Design and Redesign Process

With a clear understanding of the problems to address, the design team proceeded to create the new edit profile screen. The design process involved the following elements:

Modular Block-Based Design: We introduced a modular, block-based design approach to the screen. This design choice aimed to enhance visual consistency and allow for flexible organization of profile information. Each block had a distinct purpose, such as personal details, contact information, and privacy settings, ensuring that users could easily identify and edit the information they needed.

Layout Streamlining: The layout was reorganized to present a cleaner and more intuitive user interface. We prioritized a logical flow, ensuring that users could update their profile step by step without feeling overwhelmed.Responsive Design: Ensuring a responsive design was a top priority. This design choice meant that the edit profile screen would adapt seamlessly to various mobile devices, providing an equally smooth experience on different screen sizes and orientations.

Team Collaboration: Early in the redesign process of our mobile website (msite), the design team recognized the importance of collaborative efforts to achieve a user-friendly and visually appealing outcome. We regularly engaged in feedback sessions with cross-functional team members, including developers, content creators, and project stakeholders.

Results
Improved User Experience:
Usability tests revealed a substantial reduction in user frustration, with users finding the new edit profile screen significantly more intuitive, visually consistent, and enjoyable.

Optimized Performance:
Page load times were quantitatively measured, showing a substantial reduction in loading times, which resulted in a faster and more responsive editing experience.

Enhanced Visual Consistency:
Users provided positive feedback about the visual consistency of the screen within the new block-based framework, reinforcing the overall user experience.

Streamlined Editing Process:
Users reported a smoother experience in updating their profile information, leading to increased engagement with the feature.

Success Metrics:

The success of the enhanced edit profile screen was measured through the following key metrics:

  • User Satisfaction: User surveys and feedback channels indicated a significant increase in user satisfaction, reflecting an improved mobile user experience.
  • Load Time Reduction: Page load times were quantitatively measured, showing a reduction of 2 sec after the redesign.
  • Engagement Increase: User interactions with the edit profile screen increased by 20%, indicating that users were more actively using the feature.
  • Error Reduction: The number of user-reported errors or issues related to the edit profile screen decreased by 30%.
  • Conversion Rate: The conversion rate, in terms of users successfully updating their profiles, saw an increase of 28%.

Conclusion

The transformation of the edit profile screen within the transition from the mobile site (msite) to a block-based framework was a success. By addressing usability issues, optimizing performance, enhancing visual consistency, and streamlining the editing process, we created a user-friendly and visually appealing screen. User feedback and data indicated that this transition greatly improved the user experience, ensuring a solid foundation for future UI/UX enhancements.
